Hexo安装及部署

搭建个人blog-Hexo

目录

  • 1 快速部署Hexo
  • 2 生成静态网站命令
  • 3 连接github
  • 5 关于如何引入主题

参考链接：https://zhuanlan.zhihu.com/p/60578464

1 快速部署Hexo

npm install hexo-cli -g
hexo init [blog-name]
cd [blog-name]
npm install
hexo s  # hexo server

到此为至，博客页面可以在本地访问。

2 生成静态网站命令

hexo g  # hexo generate

3 连接github

1 项目内右键 -> Git Bash Here：

# 设置用户名和邮箱
git config --global user.name "GitHub 用户名"
git config --global user.email "GitHub 邮箱"
# 创建SSH密钥
ssh-keygen -t rsa -C "GitHub 邮箱" # 然后一路回车

2 进入C:\Users\用户名\.ssh记事本打开公钥文件id_rsa.pub ；

3 登录Github，进入Settings界面，选择SSH and GPG keys 添加密钥；

4 回到Git Bash：

ssh -T git@github.com

显示：”Hi xxx…”即连接成功。

4 创建Github Pages仓库

修改Repository name为：用户名.github.io

5 安装hexo-deployer-git：

npm install hexo-deployer-git --save

6 修改_config.yml文件末尾部分：

deploy:
  type: git
  repo: git@github.com:<Huahuatii>/<Huahuatii>.github.io.git # https://bitbucket.org/JohnSmith/johnsmith.bitbucket.io
  branch: main

完成后运行hexo d，到这里你肯定完成了在https://huahuatii.github.io/访问你的博客。
